# BUG-003 — Ürün sıralama (sort) çalışmıyor

| Alan      | Detay |
|-----------|-------|
| **Severity** | Medium |
| **User**     | problem_user |
| **Status**   | Open |

## Steps to Reproduce

1. `problem_user` ile giriş yap
2. `/inventory.html` sayfasında sıralama dropdown'ından "Price (low to high)" seç
3. Ürün fiyatlarının sıralamasını incele

## Expected

Ürünler fiyata göre artan sırada listelenmeli ($7.99 → $9.99 → $15.99 → $15.99 → $29.99 → $49.99).

## Actual

Sıralama seçeneği değiştirildiğinde ürün listesi değişmiyor. Fiyatlar rastgele sırada kalmaya devam ediyor.
Test çıktısı: `Expected: <= 9.99 / Received: 29.99`

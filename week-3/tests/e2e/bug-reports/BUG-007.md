# BUG-007 — Ürün sıralama (sort) çalışmıyor (error_user)

| Alan      | Detay |
|-----------|-------|
| **Severity** | Medium |
| **User**     | error_user |
| **Status**   | Open |

## Steps to Reproduce

1. `error_user` ile giriş yap
2. `/inventory.html` sayfasında sıralama dropdown'ından "Price (low to high)" seç
3. Ürün fiyatlarının sıralamasını incele

## Expected

Ürünler fiyata göre artan sırada listelenmeli ($7.99 → $9.99 → $15.99 → $15.99 → $29.99 → $49.99).

## Actual

Sıralama seçeneği değiştirildiğinde ürün listesi değişmiyor.
Test çıktısı: `Expected: <= 9.99 / Received: 29.99`

## Notes

BUG-003 ile aynı davranış, farklı kullanıcıda da tekrarlanıyor. Sıralama işlevi birden fazla kullanıcı için bozuk.

# BUG-008 — Login süresi 5000ms limitini aşıyor

| Alan      | Detay |
|-----------|-------|
| **Severity** | Low |
| **User**     | performance_glitch_user |
| **Status**   | Open |

## Steps to Reproduce

1. `https://www.saucedemo.com` adresine git
2. Username: `performance_glitch_user`, Password: `secret_sauce` gir
3. "Login" butonuna tıkla
4. `/inventory.html` sayfasının yüklenme süresini ölç

## Expected

Login işlemi 5000ms (5 saniye) içinde tamamlanmalı.

## Actual

Login işlemi **5172ms** sürdü — belirlenen 5000ms limitini **172ms** aşıyor.
Test çıktısı: `Expected: < 5000 / Received: 5172`

## Notes

`performance_glitch_user` için yapay gecikme uygulanıyor. Bu kullanıcı tipi performans testlerinde kasıtlı yavaş davranmak üzere tasarlanmış olsa da, login süresinin 5 saniyelik UX limitini aşması kabul edilemez.

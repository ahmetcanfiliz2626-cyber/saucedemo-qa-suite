# BUG-005 — Checkout'ta Last Name alanı otomatik siliniyor (problem_user)

| Alan      | Detay |
|-----------|-------|
| **Severity** | Critical |
| **User**     | problem_user |
| **Status**   | Open |

## Steps to Reproduce

1. `problem_user` ile giriş yap
2. Herhangi bir ürünü sepete ekle
3. Cart → Checkout'a geç
4. First Name, Last Name, Zip Code alanlarını doldur
5. "Continue" butonuna tıkla

## Expected

Tüm alanlar dolu olduğundan `/checkout-step-two.html` (overview) sayfasına geçilmeli.

## Actual

Last Name alanı doldurulmasına rağmen otomatik olarak temizleniyor. "Continue" tıklandığında `Last Name is required` hatası gösteriliyor ve sayfa `/checkout-step-one.html`'de kalıyor. Kullanıcı checkout'u tamamlayamıyor.

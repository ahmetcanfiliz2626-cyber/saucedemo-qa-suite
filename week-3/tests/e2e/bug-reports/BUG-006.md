# BUG-006 — Checkout'ta Last Name alanı otomatik siliniyor (error_user)

| Alan      | Detay |
|-----------|-------|
| **Severity** | Critical |
| **User**     | error_user |
| **Status**   | Open |

## Steps to Reproduce

1. `error_user` ile giriş yap
2. Herhangi bir ürünü sepete ekle
3. Cart → Checkout'a geç
4. First Name: `John`, Last Name: `Doe`, Zip: `12345` gir
5. Last Name alanının değerini gözlemle

## Expected

Last Name alanı `Doe` değerini korumalı.

## Actual

Last Name alanına yazıldıktan sonra alan otomatik olarak temizleniyor ve değeri `""` oluyor.
Test çıktısı: `Expected: "Doe" / Received: ""`
